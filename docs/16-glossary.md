# Glossary

## Report

A saved AI-generated business analysis containing the original prompt, generated SQL, chart configuration, result metadata, summary, and conversation history.

## Workspace

The main dashboard area where reports, charts, summaries, and tables are displayed.

## AdventureWorks Database

The Microsoft sample SQL Server database used as the analytical source of business data.

## Application Database

The internal database used by AdventureWorksAIWorkspace to store users, reports, conversations, generated SQL, tags, favorites, and export metadata.

## AI SQL Workflow

The process of transforming a natural language business question into a safe SQL query, executing it, and visualizing the result.

## Report Conversation

A chat-like history attached to a report, including the original request and follow-up refinements.

## Generated SQL Query

A SQL query generated by AI or reused by the application to answer a business question.

## Chart Definition

A saved configuration that describes how a specific chart should be rendered.

## Favorite Report

A report marked by the user as important or frequently used.

## Report Tag

A user-defined label used to organize saved reports.

## PDF Export

A downloadable PDF version of a generated report.

## Excel Export

A potential future export format for report data and summaries.
